🤖 איך להשתמש בבוט

לפני שמתחילים:

  • 📴 הפעל מצב טיסה למשך 2–3 דקות
  • 🧹 נקה את נתוני אפליקציית Wolt (ראה למטה)

לאחר מכן:

  1. פתח את הבוט בטלגרם
  2. לחץ על 🍖 קבל חשבון Wolt
  3. פתח את אפליקציית Wolt והתחבר באמצעות המייל שניתן לך
  4. המתן שהבוט ישלח את הכפתור 🩵 השלם התחברות ולחץ עליו

בחר את המכשיר שלך כדי לצפות בהוראות

📡 חשוב: יש לבצע את כל ההזמנות רק דרך אינטרנט סלולרי (4G/5G).
שימוש ב-Wi-Fi עלול לגרום לאובדן ההנחה או לחסימת החשבון.

🧹 איך לנקות את נתוני אפליקציית Wolt

⚠️ זה ימחק לחלוטין:
– את כל נתוני האפליקציה,
– את ההתחברות,
– את ההגדרות,
– את הקובצי המטמון.
האפליקציה תחזור למצב "כאילו הותקנה מחדש".

🔄 לרוב מכשירי האנדרואיד:

  1. הפעל מצב טיסה בטלפון.
  2. כנס אל הגדרות.
  3. גש לקטגוריה אפליקציות (או אפליקציות והתראות).
  4. מצא את Wolt ברשימה או חפש אותה ולחץ עליה.
  5. בחר ב-אחסון וזיכרון מטמון (או רק אחסון).
  6. לחץ על נקה אחסון (או נקה נתונים),
    ואשר את המחיקה אם מופיעה הודעת אזהרה.
  7. כבה את מצב הטיסה.

🧹 איך להסיר את אפליקציית Wolt באייפון

⚠️ זה ימחק:
– את כל נתוני המשתמש,
– את ההתחברות,
– את הגדרות האפליקציה,
– את הקבצים הזמניים.

האפליקציה תוסר לחלוטין מהמכשיר. לאחר התקנה מחדש — היא תהיה נקייה.

🍏 איך להסיר את Wolt באייפון:

  1. כנס להגדרות באייפון.
  2. גש אל כללי → אחסון iPhone.
  3. מצא את Wolt ברשימה ולחץ עליה.
  4. לחץ על מחק אפליקציה.
  5. אשר את המחיקה אם מופיעה הודעה.
  6. לאחר מכן תוכל להתקין שוב את Wolt מה-App Store.

שים לב: מחיקת האפליקציה נדרשת לפני כל התחברות חדשה דרך WoltBot.
